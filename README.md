# A-star-planner

My implementation of A* planner when I can't sleep at night xD. (jk)

Mainly for my personal learning, feel free to refer!

## Dependencies:

* OpenCV

## How-to-use:

```
#cd to repo
mkdir build
cd build
cmake ..
make
./astar-search --help
```
